# Redux Anecdotes (Part-6)

Minimal anecdote voting app using React + Redux.

Setup

1. If you cloned an external repo into this folder, remove its git metadata:

```bash
cd Part-6/redux-anecdotes
rm -rf .git
```

2. Install and run:

```bash
npm install
npm run dev
```

The app shows anecdotes and a vote button for each; votes are saved to the Redux store.
